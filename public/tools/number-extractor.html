What is Number Extractor?

Number Extractor scans pasted text and pulls out every numeric value — integers, decimals, and negatives. It is useful for quickly gathering data from reports, product listings, log files, or any text that contains mixed numeric content. The tool also computes instant statistics: count, sum, min, max, and average.

How to Extract Numbers from Text

  1. Paste any text into the input area.
  2. Choose options: include decimals, include negatives, or deduplicate.
  3. All extracted numbers appear instantly with statistics.
  4. Click "Copy as List" to copy one number per line, or "Copy as CSV" for a comma-separated format.

Frequently Asked Questions
Does it extract decimal numbers?

Yes. The "Include decimals" option is on by default and extracts numbers like 3.14, 0.5, and 99.99.

Does it extract negative numbers?

Yes. The "Include negatives" option extracts numbers prefixed with a minus sign, like -10 or -3.5.

Can I remove duplicates?

Yes. Enable the "Deduplicate" option to keep only unique values in the results.

Does it upload my text?

No. All extraction runs in your browser. No text is sent to any server.
