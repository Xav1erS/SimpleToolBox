When to use what

  • Use HEX when you need short CSS-friendly values for a design system or stylesheet.
  • Use RGB when code or a browser API expects channel values directly.
  • Use HSL when you want to change lightness or saturation more intuitively.
  • Use CMYK for print-facing conversations, not as the main format in CSS.
