What is MIME Types Reference?

MIME Types Reference maps file extensions and media types so you can find the correct Content-Type header for web responses, uploads, and assets. It is useful for API work, static file serving, storage rules, and browser compatibility debugging.

How to Use MIME Types Reference

  1. Search by file extension or MIME type string.
  2. Review the matching content type and any common usage notes.
  3. Use the mapping in server config, API headers, or upload validation.
  4. Double-check edge cases such as fonts, JSON, media, or binary files.

Frequently Asked Questions
What MIME type should I use for JSON API responses?

Use application/json with charset=utf-8.

What is application/octet-stream used for?

It is the generic binary MIME type — browsers treat it as a download prompt when served as a response.
