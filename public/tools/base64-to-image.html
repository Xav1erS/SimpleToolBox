What is Base64 to Image?

Base64 to Image decodes a Base64-encoded string or data URI back into a viewable image. It renders the image directly in the browser using an img element, displays dimensions and size, and provides download buttons for the original format and as PNG. Everything runs locally —no string or image is sent to a server.

How to Use Base64 to Image

  1. Paste a Base64 string or data URI into the input box.
  2. Click "Decode & Preview" (or press Ctrl+Enter).
  3. The image renders in the preview area with dimensions and size info.
  4. Click a download button to save the image file.

Frequently Asked Questions
What is a Base64 image string?

A Base64 image is a binary image file encoded as ASCII characters. A data URI wraps it as data:image/png;base64,... so a browser can render it directly from the string without loading an external file.

How do I get the correct Base64 string?

Paste either a full data URI (starting with "data:image/...") or a raw Base64 string (the characters after the comma in a data URI). The tool auto-detects the format and renders the image.

What image formats are supported?

Any image format your browser can render: JPEG, PNG, WebP, GIF, SVG, BMP, and ICO. The tool detects the format from the Base64 magic bytes.

Can I download the decoded image?

Yes. Click "Download original format" to save the file in its native format. For raster images, "Download as PNG" converts via a canvas element.
