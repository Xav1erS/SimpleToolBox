What is PDF Page Counter?

PDF Page Counter is a browser-based tool that instantly reads the page count and document metadata from any PDF file. It uses PDF.js to parse the file locally —your PDF is never sent to a server. In seconds you get the total page count displayed prominently, alongside all available document properties: title, author, subject, creator application, producer library, and both creation and modification timestamps.

How to Use PDF Page Counter

  1. Drop a PDF file onto the upload area or click to browse and select one.
  2. The total page count is shown immediately as a large number.
  3. Scroll down to see the full metadata grid: file name, size, title, author, subject, creator, producer, creation date, and modification date.
  4. Click "Count another file" to reset and analyse a different PDF.

Frequently Asked Questions
Is my PDF uploaded to a server?

No. The tool runs entirely inside your browser using PDF.js. Your file is never sent to any server and never leaves your device.

What metadata fields are shown?

The tool displays Title, Author, Subject, Creator, Producer, Creation Date, and Modification Date —all read from the PDF's internal document information dictionary. Fields not present in the file are shown as a dash.

Does it work with large PDFs?

Yes. The tool only loads the document structure and metadata, not the full page rendering data, so even large PDFs are processed quickly with minimal memory use.

What PDF versions are supported?

PDF.js supports the full range from PDF 1.x through PDF 2.0, including password-protected files (you will be prompted to enter the password).
