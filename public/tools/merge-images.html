What is Merge Images?

Merge Images combines multiple photos or screenshots into one output image. It is useful for step-by-step guides, support reports, social carousels, and any workflow where one combined image is easier to share than several separate files.

How to Use Merge Images

  1. Upload at least two images into the tool.
  2. Choose a vertical or horizontal merge layout.
  3. Adjust the gap and background color if needed.
  4. Preview the combined result and download the merged PNG.

Frequently Asked Questions
Can I merge images vertically and horizontally?

Yes. Switch between vertical and horizontal layout depending on whether you want a stacked strip or a side-by-side image.

Can I control spacing between images?

Yes. Use the gap slider to add no spacing or a larger gutter between each image.

What output format does it download?

This version downloads the merged result as a PNG so the combined layout stays intact.

Are the uploaded images sent anywhere?

No. Everything runs locally in your browser.
