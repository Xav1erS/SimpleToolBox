What is PDF Compressor?

PDF Compressor is a browser-based tool that reduces PDF file size by re-rendering each page as a compressed JPEG image and rebuilding the document with jsPDF. Three quality presets let you balance file size against visual clarity. Because everything runs inside your browser using PDF.js and jsPDF, your file is never sent to any server.

How to Use PDF Compressor

  1. Drop a PDF file onto the upload area or click to browse and select one.
  2. Choose a quality preset: High (150 DPI), Medium (96 DPI), or Low (72 DPI).
  3. Click the Compress PDF button and wait for processing to finish.
  4. Review the before and after file sizes and savings percentage.
  5. Click Download Compressed PDF to save the result.

Frequently Asked Questions
Is my PDF uploaded to a server?

No. Everything runs inside your browser using PDF.js and jsPDF. Your file never leaves your device.

Why does the file size increase sometimes?

Very low-resolution source PDFs or PDFs that are already heavily compressed may not reduce further. In those cases, switching to the High quality preset can help, or the file is already near its minimum size.

Does compression preserve the text layer?

No. Each page is rasterized to a JPEG image, so the text layer is not preserved. Use the High quality preset to keep text legible.

What is the maximum file size?

There is no hard limit, but very large PDFs may take longer to process and use more browser memory. For best performance keep files under 50 MB.
