      Frequently Asked Questions
      

        
What is the difference between AA and AAA?
AA is the standard compliance level required by most regulations. AAA is enhanced — not required for all content but improves readability.

        
Does this check non-text contrast?
WCAG 1.4.11 requires 3:1 contrast for UI components and graphical elements — this checker covers that requirement.

        
What does WCAG Color Contrast Checker - Batch AA/AAA Test do?
WCAG Color Contrast Checker - Batch AA/AAA Test runs entirely in your browser and helps you complete this task without uploading data to a server.

        
Is WCAG Color Contrast Checker - Batch AA/AAA Test free to use?
Yes. This tool is free to use and does not require sign-up.
