ASCII Character Categories
  • Control chars (0–31): non-printable terminal control codes (NUL, TAB, LF, CR, ESC...)
  • Printable chars (32–126): space, digits, letters, punctuation
  • DEL (127): delete control character

Frequently Asked Questions
Is extended ASCII (128–255) included?

This table shows standard 7-bit ASCII (0–127). Extended ASCII (Latin-1, Code Page 437) varies by system.

What is the NULL character (0) used for?

ASCII 0 (NUL) is a string terminator in C and embedded in file formats as a field separator.
