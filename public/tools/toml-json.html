TOML (Tom's Obvious Minimal Language) is a config file format used by Cargo, uv, pyproject.toml, and many CLI tools. Use this converter to inspect TOML configs as JSON or generate TOML from a JSON object. Note: TOML dates are preserved as ISO strings in JSON output.
