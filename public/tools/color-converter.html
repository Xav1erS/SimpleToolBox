About Color Formats
  • HEX — Most common in web / CSS / HTML. Compact 6-character hexadecimal notation (e.g. #FF5733).
  • RGB — Used in screen-based programming, CSS, and canvas APIs. Each channel (Red, Green, Blue) ranges 0–255.
  • HSL — Hue, Saturation, Lightness. More intuitive for humans — easy to tweak brightness or tone a color.
  • CMYK — Cyan, Magenta, Yellow, Key (Black). Used in print design and professional publishing workflows.
  • Edit any field and all other formats update automatically. Supports both short-form (255, 87, 51) and full-form (rgb(255, 87, 51)) input.

Frequently Asked Questions
Does it support CSS oklch() and lab()?

This tool converts between traditional color spaces. CSS oklch/lab require a dedicated perceptual color converter.

How accurate is the CMYK conversion?

The conversion uses the sRGB → CMYK formula. For print production, use a ICC-profile-aware tool.
