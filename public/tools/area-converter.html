What is an Area Converter?

An area converter translates a 2D surface measurement between different units. Common metric units include square millimeters (mm2), square centimeters (cm2), square meters (m2), and hectares (ha). Common imperial units include square inches (in2), square feet (ft2), and acres. This tool converts between all eight units simultaneously.

How to Convert Area

  1. Type a value into any field (mm2, cm2, m2, km2, in2, ft2, acres, or ha).
  2. All other fields update instantly.
  3. Use quick example buttons for common reference areas.
  4. Click Clear to reset all fields.

Frequently Asked Questions
How many square feet in an acre?

1 acre = 43,560 square feet.

How many acres in a hectare?

1 hectare = 2.47105 acres.

How many square meters in a hectare?

1 hectare = 10,000 square meters (a 100 m x 100 m square).

How many square feet in a square meter?

1 square meter = 10.7639 square feet.
