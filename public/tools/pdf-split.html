What is PDF Splitter?

PDF Splitter is a browser-based tool that lets you extract specific pages or split an entire PDF into individual files вАФall without uploading anything to a server. Choose "By Range" to define custom page groups (e.g. 1-3, 5, 7-9), or choose "Every Page" to get each page as its own PDF in a ZIP archive.

How to Use PDF Splitter

  1. Drop a PDF file onto the upload area or click to browse and select one.
  2. Select a split mode: By Range to specify custom page groups, or Every Page to extract all pages individually.
  3. For By Range mode, type your ranges in the text field вАФfor example 1-3, 5, 7-9.
  4. Click Split PDF and wait for processing to finish.
  5. Download each output PDF individually, or use the ZIP button to get all files at once.

Frequently Asked Questions
Is my PDF uploaded to a server?

No. The entire process runs inside your browser using PDF.js and jsPDF. Your file never leaves your device.

How do I specify page ranges?

Enter ranges like 1-3, 5, 7-9 separated by commas. Each comma-separated entry produces one output PDF.

What happens when I choose Every Page?

Each page becomes a separate PDF file. All files are bundled into a single ZIP archive for easy download.

Does it preserve text and vector content?

No. Pages are rasterized to JPEG images via canvas before being embedded in the output PDF. The output PDFs contain high-resolution images of the pages, not selectable text.
