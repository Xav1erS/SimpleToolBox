Box Shadow Parameters
  • X/Y Offset — shifts the shadow horizontally and vertically
  • Blur — how soft/blurred the shadow edge is
  • Spread — expands or contracts the shadow size
  • Inset — renders shadow inside the element
Frequently Asked Questions
Can I add multiple shadows?

Yes. Click 'Add Layer' to stack multiple box-shadow values, combined with commas in the output CSS.

How do I use the generated CSS?

Click 'Copy CSS' and paste the box-shadow property directly into your CSS file or style block.
