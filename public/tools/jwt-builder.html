What is JWT Builder?

JWT Builder creates signed JSON Web Tokens in the browser using HMAC algorithms such as HS256, HS384, and HS512. It is useful for generating sample tokens, testing claim sets, and validating how your application handles roles, expiry, and custom payload fields.

How to Use JWT Builder

  1. Choose the HMAC algorithm you want to use.
  2. Enter or edit the payload claims you need in the token.
  3. Provide the signing secret and build the token.
  4. Copy the generated JWT and test it in your local app or API flow.

Frequently Asked Questions
Can I use RS256 (asymmetric) signing?

No. This builder is focused on HMAC signing modes such as HS256, HS384, and HS512. Use a dedicated JWT tool or backend library for RSA and EC key-based signing.

How do I decode an existing JWT?

Use the JWT Decoder tool to inspect any JWT without a secret.

What does JWT Builder / Generator - Create JSON Web Tokens do?

JWT Builder / Generator - Create JSON Web Tokens runs entirely in your browser and helps you complete this task without uploading data to a server.

Is JWT Builder / Generator - Create JSON Web Tokens free to use?

Yes. This tool is free to use and does not require sign-up.
