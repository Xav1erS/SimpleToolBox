What is ASCII Table Reference?

ASCII Table Reference lists standard 7-bit ASCII characters along with their decimal, hexadecimal, octal, and binary codes. It is useful for low-level debugging, teaching text encoding basics, and checking character values in logs, code, and protocols.

How to Use ASCII Table Reference

  1. Search by character, code value, or description.
  2. Review the matching row across decimal, hex, octal, and binary columns.
  3. Use the result to compare values in code, docs, or logs.
  4. Copy the relevant code or character for your workflow.

Frequently Asked Questions
Is extended ASCII (128–255) included?

This table shows standard 7-bit ASCII (0–127). Extended ASCII (Latin-1, Code Page 437) varies by system.

What is the NULL character (0) used for?

ASCII 0 (NUL) is a string terminator in C and embedded in file formats as a field separator.
