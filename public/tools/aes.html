AES Encrypt / Decrypt

Encrypt and decrypt text using AES-256-GCM with a passphrase. Runs entirely in your browser.

Note: Uses AES-256-GCM with PBKDF2 key derivation (100,000 iterations, SHA-256). A random salt (16 bytes) and IV (12 bytes) are generated for each encryption. The output is a single Base64 string containing salt + IV + ciphertext. Your passphrase is never uploaded — all processing happens in your browser. To decrypt, you must use the same passphrase that was used to encrypt.
