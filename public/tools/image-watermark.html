What is Image Watermark?

Image Watermark lets you overlay custom text on any photo directly in your browser. You can control the font, size, style, color, opacity, and exact position of the watermark across a 3×3 position grid. This is useful for copyright protection, branding, and draft labeling without installing any software.

How to Use Image Watermark

  1. Upload the image you want to watermark.
  2. Type your watermark text in the Text field.
  3. Adjust font size, style, color, and opacity as needed.
  4. Click a position button to place the watermark where you want it.
  5. Download the watermarked PNG.

Frequently Asked Questions
What image formats are supported?

You can upload JPG, PNG, WebP, and GIF images. The result is always downloaded as a PNG.

Can I change the watermark position?

Yes. There is a 3×3 position grid covering all nine corners and edges. Click any position to move the watermark.

How do I make the watermark semi-transparent?

Use the Opacity slider. Lower values produce a more transparent watermark that blends into the image.

Is the image uploaded to a server?

No. All watermark processing happens locally in your browser using the Canvas API.
