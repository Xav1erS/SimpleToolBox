What is PDF Merger?

PDF Merger is a browser-based tool that combines multiple PDF files into a single PDF document. Upload as many PDFs as you need, drag the rows to set the order, then click Merge PDFs to generate a unified file ready to download. Because everything runs inside your browser using PDF.js and jsPDF, no file is ever sent to a server.

How to Use PDF Merger

  1. Drop two or more PDF files onto the upload area, or click to browse and select them.
  2. Drag the file rows to arrange the order in which pages will appear in the output.
  3. Click Merge PDFs and wait for the progress bar to complete.
  4. Click Download merged.pdf to save the combined file.
  5. Use Merge another set to start over with a fresh set of files.

Frequently Asked Questions
Are my PDFs uploaded to a server?

No. Everything runs entirely inside your browser. Your files never leave your device.

How many PDFs can I merge?

There is no hard limit. You can merge as many files as you like, though very large or numerous PDFs may use significant browser memory. For best results, keep total file size under 100 MB.

Is the page order preserved?

Yes. Pages from each file appear in order. Drag the file rows to reorder files before you click Merge PDFs.

Does it preserve text and links from the original PDFs?

No. Each page is rasterized to an image and embedded in the output PDF. Text selection and hyperlinks are not preserved. This tool is best suited for image-heavy or scanned documents.
