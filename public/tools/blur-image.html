Frequently Asked Questions
Can I blur only part of the image (e.g., just a face)?

This tool applies blur to the entire image. For selective/regional blurring, you would need image editing software (e.g., GIMP, Photoshop, Canva) that supports layer masks or selection tools. However, this tool is useful for blurring background images entirely or applying an artistic blur to the whole photo.

What blur radius should I use for privacy purposes?

For effective privacy blur that obscures recognizable details (faces, license plates, screen content), a radius of 20—0px is recommended. The "Privacy" preset at 30px provides strong anonymization for most images.

Does blurring an image reduce file size?

Often yes —blurred images have less detail and more uniform color areas, which makes JPEG compression more effective. A heavily blurred image can be 30—0% smaller than the original at the same quality setting.

Can I undo the blur?

Once downloaded, blur cannot be reversed —blurring permanently destroys the image detail in those areas. The original file in your browser is not modified; just reload the page and upload the original again.
