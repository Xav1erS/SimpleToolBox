What is a Length Converter?

A length converter translates a distance or size value between different measurement systems. The metric system uses millimeters, centimeters, meters, and kilometers. The imperial/US customary system uses inches, feet, yards, and miles. This tool converts between all eight units simultaneously.

How to Use Length Converter

  1. Type a value into any field (mm, cm, m, km, in, ft, yd, or mi).
  2. All other fields update instantly.
  3. Click a quick example button to load a common reference length.
  4. Click Clear to reset all fields.

Frequently Asked Questions
How many centimeters are in an inch?

1 inch = 2.54 centimeters exactly.

How many feet are in a meter?

1 meter = 3.28084 feet.

How many kilometers in a mile?

1 mile = 1.60934 kilometers.

How many inches in a foot?

1 foot = 12 inches.
