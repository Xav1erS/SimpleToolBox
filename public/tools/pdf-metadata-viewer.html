What is PDF Metadata Viewer?

PDF Metadata Viewer is a browser-based tool that reads and displays all embedded metadata from any PDF file. It uses PDF.js to parse the file entirely in your browser —your PDF is never sent to a server. In seconds you see the complete document information: title, author, subject, keywords, creator application, producer library, creation and modification timestamps, PDF version, page count, file size, and the first page's dimensions and orientation.

How to Use PDF Metadata Viewer

  1. Drop a PDF file onto the upload area or click to browse and select one.
  2. The Document Info section shows title, author, subject, keywords, creator, producer, and both dates.
  3. The File Info section shows PDF version, page count, file size, page dimensions, and orientation.
  4. Click "Copy All as Text" to copy the metadata as key:value lines for pasting elsewhere.
  5. Click "Inspect another file" to reset and analyse a different PDF.

  • Verify the author and creation date of a received PDF document

  • Confirm page dimensions before printing or sending to a print shop

Frequently Asked Questions
Is my PDF uploaded to a server?

No. The tool runs entirely inside your browser using PDF.js. Your file is never sent to any server and never leaves your device.

What metadata fields does this tool show?

The tool shows Title, Author, Subject, Keywords, Creator (authoring application), Producer (PDF library), Creation Date, Modification Date, PDF Version, page count, file size, page dimensions, and orientation —all read from the PDF's internal document information dictionary.

Why are some metadata fields empty?

PDF metadata fields are optional. Many PDFs are exported without a title or author filled in. Fields not present in the file are displayed as a dash. The PDF version, page count, file size, and page dimensions are always available regardless of the document's metadata.

What PDF versions are supported?

PDF.js supports the full range from PDF 1.x through PDF 2.0, including password-protected files (you will be prompted to enter the password).
