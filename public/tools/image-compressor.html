What is Image Compression?

Image compression reduces the file size of an image by discarding redundant or less-important pixel data. Lossy formats like JPEG and WebP achieve high compression by slightly reducing visual quality, while PNG uses lossless compression to preserve every pixel. This tool uses your browser's Canvas API to re-encode images at your chosen quality level — nothing is uploaded to any server.

How to Compress an Image

  1. Drop an image onto the preview area, or click to browse and select a file.
  2. Choose an output format: JPEG (best for photos), WebP (best compression-to-quality ratio), or PNG (lossless).
  3. Drag the quality slider — lower values produce smaller files; higher values preserve more detail.
  4. Watch the File Size panel update in real time to see the savings.
  5. Click Download Compressed to save the result.

JPEG vs WebP vs PNG — Which to Choose?

  • JPEG — Best for photographs. Widely supported by all browsers, email clients, and social media. Good compression at 70–85% quality.
  • WebP — Google's modern format. Typically 25–35% smaller than JPEG at equivalent quality. Supported by all modern browsers.
  • PNG — Lossless. Best for logos, screenshots, and graphics with transparency. File size is larger than JPEG/WebP for photos.

Frequently Asked Questions

Is the image uploaded to a server?

No. All compression happens entirely in your browser using the HTML5 Canvas API. Your images never leave your device.

What quality setting should I use?

For most web images, 75–85% quality strikes the best balance between file size and visual fidelity. For thumbnails or previews, 60–70% is acceptable. Below 50% visible artifacts may appear in photographs.

Why does PNG output have no quality effect?

PNG is a lossless format — quality does not apply. For significant size reduction, switch to JPEG or WebP, which are lossy formats.

Can I compress multiple images at once?

This tool processes one image at a time. For batch compression you would need a desktop application such as ImageOptim or Squoosh CLI.
