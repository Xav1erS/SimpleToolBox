Frequently Asked Questions
Is deduplication case-sensitive?

Yes — "Apple" and "apple" are treated as different lines. Use the lowercase conversion first if you want case-insensitive deduplication.

Can I filter with a regex pattern?

Yes — the filter field accepts regular expressions.

Understand Line Tools (examples, use cases & FAQs)
Example

Sorting a fruit list alphabetically:

Input banana apple cherry
Sort A→Z apple banana cherry

The Sort operation orders lines alphabetically (A→Z or Z→A) in a single click.

Supported Operations
  • Sort A→Z / Z→A — alphabetically order lines ascending or descending.
  • Reverse — flip the order of lines (last line becomes first).
  • Deduplicate — remove repeated lines, keeping the first occurrence.
  • Shuffle — randomise the order of lines.
  • Filter — keep only lines that match (or do not match) a keyword or pattern.
  • Number Lines — prepend each line with a sequential line number.
Why Use This Tool
  • All operations in one place — sort, filter, and deduplicate without switching tools.
  • Works on any list — process CSV columns, log lines, word lists, or code snippets.
  • Case-insensitive sorting option — mix upper and lower case without unexpected ordering.
  • Live output — see results immediately as you adjust options.
  • Free & private — no data is uploaded; everything runs in your browser.
Frequently Asked Questions

Does deduplicate keep the first or last occurrence?

By default it keeps the first occurrence of each repeated line and removes subsequent duplicates. The original order of unique lines is preserved.

Can I filter lines using a regular expression?

Yes. The Filter operation accepts plain keywords as well as regular expressions, giving you precise control over which lines to keep or discard.

How many lines can it handle?

The tool handles tens of thousands of lines without issue because all processing runs in the browser's JavaScript engine with no upload or server round-trip.

Is my text stored or shared anywhere?

No. All processing happens locally in your browser. No data is sent to any server.
