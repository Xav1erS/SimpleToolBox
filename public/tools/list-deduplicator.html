What is List Deduplicator?

List Deduplicator removes repeated items from delimited lists such as comma-separated values, newline lists, tab-separated rows, and semicolon-separated fields. It is useful when copied data is not formatted as one item per line but still needs to be cleaned quickly.

How to Use List Deduplicator

  1. Paste the list and choose the delimiter used in the input.
  2. Choose the output delimiter you want for the cleaned result.
  3. Turn on trimming or ignore case if you want stricter matching.
  4. Copy the unique list after the output updates.

Frequently Asked Questions
What input separators does it support?

It supports newline, comma, semicolon, and tab separated lists.

Can I output the result in a different delimiter?

Yes. You can parse one delimiter and export the unique list in another.

Does it trim spaces around each item?

Yes. Trim each item is enabled by default so values like Apple and Apple count as the same item more often.

Is the list processed locally?

Yes. Everything runs in your browser and no list data is uploaded.
