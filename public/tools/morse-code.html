Morse code was developed in the 1830s–1840s by Samuel Morse and Alfred Vail for the electric telegraph. Each letter or digit is represented by a sequence of dots (.) and dashes (-). Within a word, characters are separated by a single space; words are separated by a / (representing a longer pause — traditionally 3 dot-lengths of silence). All processing is done entirely in your browser — no data is transmitted.
