What is a Time Converter?

A time converter translates a duration between different time units. Common units range from milliseconds (ms) used in computing, to seconds, minutes, hours, and days used in everyday life, up to weeks, months, and years for longer periods. This tool uses the average Gregorian year (365.2425 days) for month and year conversions.

How to Convert Time

  1. Type a duration into any field (ms, s, min, h, day, wk, mo, or yr).
  2. All other fields update instantly.
  3. Use quick example buttons for common reference durations.
  4. Click Clear to reset all fields.

Frequently Asked Questions
How many seconds in a day?

1 day = 86,400 seconds (24 hours x 60 minutes x 60 seconds).

How many seconds in a year?

Approximately 31,557,000 seconds (365.25 days x 86,400).

How many weeks in a year?

Approximately 52.18 weeks in a year (365.25 / 7).

What month length does this tool use?

Average Gregorian month: 365.2425 / 12 = 30.436875 days = 2,629,746 seconds.
