What is CSS Gradient Generator?

CSS Gradient Generator is a visual tool for building CSS linear and radial gradients. You add color stops, set their positions on a 0–100% scale, choose an angle for linear gradients, and see the result live. The final CSS property is ready to paste into any stylesheet.

How to Use CSS Gradient Generator

  1. Choose Linear or Radial gradient type.
  2. For linear gradients, set the angle using the slider.
  3. Edit the default color stops by clicking the color pickers and adjusting positions.
  4. Add more color stops with the "Add color stop" button if needed.
  5. Click "Copy CSS" to copy the gradient property to your clipboard.

Frequently Asked Questions
What gradient types are supported?

Linear gradients with a custom angle (0–360°) and radial gradients (circle shape) are both supported.

Can I add more than two color stops?

Yes. Click "Add color stop" to add as many stops as you need. Each stop has an independent color and position.

How do I control the direction of a linear gradient?

Use the Angle slider. 0° is top to bottom, 90° is left to right, and 135° runs diagonally.

Is the output cross-browser compatible?

Yes. The tool outputs standard CSS using the linear-gradient() and radial-gradient() functions which are supported by all modern browsers.
