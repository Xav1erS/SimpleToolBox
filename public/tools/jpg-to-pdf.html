What is JPG to PDF?

JPG to PDF is a free browser tool that converts one or more JPEG photos into a single PDF document. It runs entirely client-side using jsPDF —your photos are never uploaded to any server. Choose page size, margin, and orientation, then download the result immediately.

How to Use JPG to PDF

  1. Drop your JPG files onto the upload area, or click to browse and select them.
  2. Choose a page size (A4, Letter, or Fit to Image), orientation, and margin.
  3. Click Convert to PDF —the file downloads automatically.

Frequently Asked Questions
Are my photos uploaded to a server?

No. All conversion happens entirely in your browser using jsPDF. Your images are never sent to any server.

Can I convert multiple JPG files at once?

Yes. Select or drop multiple JPG files and each image will become one page in the resulting PDF. The download will be named after your single file if you convert just one, or 'images.pdf' for multiple files.

What page size should I choose?

A4 is the international standard used in most countries. Letter is the North American standard (8.5×11 inches). Choose "Fit to Image" if you want each PDF page to exactly match the pixel dimensions of each photo.

How is this different from the generic Image to PDF tool?

This tool accepts only JPG/JPEG files and is optimised for the common task of converting photos. It uses auto-detect orientation based on each image's aspect ratio and offers a streamlined interface. The generic Image to PDF tool also supports PNG, WebP, and GIF with manual reordering.
