What is JSONPath Tester?

Tests JSONPath expressions against JSON data in real time. Supports dot notation, bracket notation, filter expressions, recursive descent, and array slicing per the RFC 9535 JSONPath specification.

How to Use JSONPath Tester

  1. Paste your JSON in the top panel.
  2. Enter a JSONPath expression (e.g., $.store.books[*].title).
  3. See matching nodes highlighted and listed instantly.
  4. Copy the results.

Understand JSONPath Tester ▾
Example

JSON: {"users":[{"name":"Alice","age":30},{"name":"Bob","age":17}]}

Path: $.users[?(@.age >= 18)].name

Result: ["Alice"]

JSONPath Syntax Reference
  • $ root element
  • .key child key
  • [*] all array elements
  • [0] first element, [-1] last element
  • ..key recursive descent
  • [?(@.age > 18)] filter expression
  • [1:3] array slice
Why Use This Tool
  • Test JSONPath expressions before embedding in code
  • Debug complex nested JSON structures
  • Works with any API response or config file
Frequently Asked Questions
Is JSONPath the same as jq?

No — jq is a separate query language. JSONPath is standardized in RFC 9535 and supported by many libraries.

Can I use multiple expressions at once?

Currently one expression at a time — results show all matching nodes.
