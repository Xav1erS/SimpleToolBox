What is Text List to Array?

Text List to Array converts a plain text list —one item per line —into an array literal in the programming language of your choice. It supports JavaScript, Python, PHP, Ruby, and JSON, with options for quote style and whitespace handling.

How to Use Text List to Array

  1. Paste your list (one item per line) into the input area.
  2. Select the target language: JavaScript, Python, PHP, Ruby, or JSON.
  3. Choose single or double quotes.
  4. Toggle 'Trim whitespace' and 'Skip empty lines' as needed.
  5. Click Copy to copy the array literal.

Frequently Asked Questions
Which languages are supported?

JavaScript, Python, PHP, Ruby, and JSON. The array syntax differs slightly between languages —for example Python and JavaScript use the same bracket syntax but PHP and Ruby differ.

Can I choose single or double quotes?

Yes. Use the Quote dropdown to switch between single and double quotes. JSON always uses double quotes regardless of the setting.

What happens to empty lines?

By default empty lines are skipped. Uncheck 'Skip empty lines' to include them as empty string items.

Does it handle special characters?

Yes. Quotes and backslashes inside items are automatically escaped for the selected language.
