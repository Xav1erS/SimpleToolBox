What is JSON ↔ CSV Converter?

Converts JSON arrays to CSV spreadsheet format and CSV data back to JSON. Automatically detects column headers from JSON keys and handles nested objects by flattening them.

How to Use JSON ↔ CSV Converter

  1. Paste a JSON array or CSV data.
  2. Click Convert.
  3. Choose delimiter: comma, tab, or semicolon.
  4. Copy or download the converted output.

Handling Complex JSON

Flat arrays convert directly. Nested objects are flattened with dot notation (address.city becomes a column). Arrays within objects are serialized as JSON strings. Null values become empty CSV cells. All column headers are auto-detected from the first record.

Why Use This Tool
  • No Excel macros or Python scripts needed for quick conversion
  • Handles nested JSON automatically
  • Works for API output → spreadsheet workflows
Frequently Asked Questions
Does it handle arrays nested inside objects?

Nested arrays are converted to JSON string columns. For complex nesting, consider pre-processing your JSON first.

What CSV delimiters are supported?

Comma, semicolon, tab, and pipe (|) delimiters are supported.
