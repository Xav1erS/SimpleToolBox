How chmod octal notation works

Each of the three digits represents one class (owner / group / others). The digit is the sum of r=4, w=2, x=1 for that class.

7=111=rwx (read + write + execute)
6=110=rw- (read + write)
5=101=r-x (read + execute)
4=100=r-- (read only)
3=011=-wx (write + execute)
2=010=-w- (write only)
1=001=--x (execute only)
0=000=--- (no permissions)

Understand Chmod Permission Calculator ▾
Example

Checked: Owner rwx, Group r-x, Others r-x → 755 (rwxr-xr-x)

Command: chmod 755 filename

Common values: 644 (rw-r--r--), 600 (rw-------), 400 (r--------)

Common Permission Values
  • 755 (rwxr-xr-x): standard for executable files and directories
  • 644 (rw-r--r--): standard for regular files
  • 600 (rw-------): private keys and secrets
  • 777 (rwxrwxrwx): world-writable — use with caution
  • 400 (r--------): read-only by owner

Frequently Asked Questions
Does it support special bits (setuid, sticky)?

Yes — setuid (4000), setgid (2000), and sticky bit (1000) can be toggled.

What does execute permission mean for a directory?

Execute on a directory means the ability to cd into it and access its contents.
