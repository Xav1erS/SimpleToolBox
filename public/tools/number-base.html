What is Number Base Converter?

This tool converts integer values between binary, octal, decimal, and hexadecimal in real time. It is useful for programming, debugging, low-level data inspection, and permission math.

How to Use Number Base Converter

  1. Type a value into any base field.
  2. The matching values in the other bases update instantly.
  3. Use the bit info panel to inspect bit length and unsigned views.
  4. Copy any representation you need for code, logs, or documentation.

Arbitrarily large integers are supported using JavaScript BigInt.

Positive integers are supported. Negative numbers and floating-point values require a separate converter.
